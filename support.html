3D Nozzle MM to IN Helper Support

Nozzle conversion utility  ·  Version 1.0

Thanks for using 3D Nozzle MM to IN Helper. If something looks wrong, these quick checks usually solve it.

The number will not parse

Use only digits and one decimal separator. The app accepts your device locale and also tolerates the common alternate separator.

A preset did not match what I expected

The preset row is built around common nozzle diameters in millimeters. If the input side is set to inches, the preset converts the matching millimeter preset into inches automatically.

A parts link looks outdated

The Parts tab opens external Amazon pages and availability may change over time. If a link becomes stale, send the title that failed and we will refresh it in the next update.
